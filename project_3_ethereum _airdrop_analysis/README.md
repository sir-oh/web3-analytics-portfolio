**Project 3 — Ethereum Airdrop Retention & Wallet Behavior**
**Overview**

This project analyzes the Arbitrum (ARB) airdrop to evaluate whether token distribution led to long-term user engagement or short-term farming behavior.

The focus is on wallet retention, behavior segmentation, and capital concentration.

**Problem Statement**

Airdrops are costly growth mechanisms, but their effectiveness depends on whether recipients remain active users after distribution.

**Data**

Ethereum ERC-20 transfer events

Airdrop distribution data

Wallet-level transaction history

**Methodology**

Identified airdrop recipient wallets

Tracked post-airdrop token holding behavior

Built retention curves over time

Segmented wallets into whale vs retail cohorts

Analyzed differences in behavior across segments

**Key Insights**

A significant portion of wallets exited shortly after receiving tokens

Smaller wallets showed higher relative retention than large recipients

Airdrop size strongly influenced post-distribution behavior

**Tools**

Dune SQL · Ethereum on-chain data · Dashboard visualizations
